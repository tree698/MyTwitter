## :white_check_mark:&nbsp;주요 특징
- MVC 패턴 적용
- validation, sanitization 적용
- Restful APIs 서비스 적용
- Json Web Token(JWT)을 이용한 인증 기능 적용
- socket-io를 활용한 실시간 채팅 기능
- MySQL과 Sequelize ORM 사용
